║              Безопасный VPN клиент для Windows                ║
╚═══════════════════════════════════════════════════════════════╝

📁 СОДЕРЖИМОЕ:
   • KampusVPN.exe - основное приложение
   • bin/sing-box.exe - VPN ядро (sing-box 1.12.x)

🚀 ЗАПУСК:
   1. Распакуйте архив в любую папку
   2. Запустите KampusVPN.exe
   3. При первом запуске добавьте подписку VPN или WireGuard конфиг

⚙️ ФУНКЦИИ:
   • Поддержка подписок (VLESS, VMess, Trojan, Shadowsocks, Hysteria2, TUIC)
   • WireGuard конфигурации для корпоративных сетей
   • Несколько профилей подключения
   • QR-код сканер для добавления подписок
   • Автозапуск при старте системы
   • Автообновление приложения
   • Светлая/Тёмная тема
   • Русский/Английский язык

📋 ТРЕБОВАНИЯ:
   • Windows 10/11 (64-bit)
   • WebView2 Runtime (обычно уже установлен)

🔧 РЕШЕНИЕ ПРОБЛЕМ:
   • Если приложение не запускается - установите WebView2:
     https://developer.microsoft.com/en-us/microsoft-edge/webview2/
   
   • Если VPN не подключается - проверьте что антивирус 
     не блокирует sing-box.exe
